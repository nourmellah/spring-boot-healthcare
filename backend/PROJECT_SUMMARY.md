## 🗄️ Database Relationships

```
User (Base)
├── Patient (extends User)
│   ├── 1:N → Appointments
│   ├── 1:N → Prescriptions
│   ├── 1:N → LabResults
│   ├── 1:N → EmergencyAlerts
│   ├── 1:N → VitalSigns
│   └── N:1 → Universite
│
├── Doctor (extends User)
│   ├── 1:N → Appointments
│   ├── 1:N → Prescriptions
│   ├── 1:N → ConsultationNotes
│   └── N:1 → Specialty
│
Appointment
├── N:1 → Patient
├── N:1 → Doctor
├── 1:1 → Prescription
└── 1:1 → ConsultationNote

Prescription
├── N:1 → Patient
├── N:1 → Doctor
├── 1:1 → Appointment
└── 1:N → PrescriptionMedicines
    └── N:1 → Medicine

EmergencyAlert
├── N:1 → Patient
└── N:1 → Doctor (optional)

LabResult
├── N:1 → Patient
└── N:1 → Doctor (optional)

VitalSign
└── N:1 → Patient

Notification
└── N:1 → User (userId)

AuditLog
└── N:1 → User (userId)
```


### 2. Create Initial Data
```bash

# 2. Register Doctor

# 3. Register Patient
POST /api/patients

# 4. Book Appointment
POST /api/appointments
```
